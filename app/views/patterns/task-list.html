Task list

Help users know what they need to complete or provide within their application, whilst keeping track of what has already been completed, and costs established.

We found users were skipping past the 'Before you start section' so renamed this section. This is still in test.

Due to the number of sections to our bespoke application process, we found that these six sections to our task list made it easier to navigate.

Cost and activities summary above the task list helped remind users of the cost and activities selected pre task list. Having a link to allow a change to the chosen activities at this section has tested okay so far, and was introduced as a response to users wanting to go back to change activities.

When to use this pattern

This pattern should be used for a complex application process. This differs from the Design System pattern only by section headings, and the activities and cost details placed above the list.

Styles for this patterns can be found in assets/sass/patterns.
