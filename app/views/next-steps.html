Next steps boxout

Help users know what to do next if there is no clear onward action, for example if you have sent them an email.

When to use this pattern

Use this pattern when user research shows that users are unclear what they should be doing next. This commonly occurs when there is a break in the process, for example they need to go and get the docuemnts you told them they'd need in the 'before you start' section.

Styles for this patterns can be found in assets/sass/patterns.

Next steps

  1. Find the email with the subject "You requested access to renew your waste exemptions"
  2. Open email and follow the instructions to sign in
  3. If you cannot see an email from us, please check your Bulk/Spam folder
  4. If you need us to re-send the email, see the "Need help?" area
